# CIFAR-10 Image Classifier

Built a convolutional neural network using PyTorch to classify images from the CIFAR-10 dataset as part of a Udacity nanodegree project.

## What it does
Classifies images into 10 categories: plane, car, bird, cat, deer, dog, frog, horse, ship, and truck.

## Results
Achieved 73% accuracy on the test set, beating the 70% benchmark.

## How to run
Open the notebook and run all cells. Requires PyTorch and torchvision.
